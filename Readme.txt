This plugin can changes the logo & background image of the login form.

== Description ==
Just Install the plugin, then you can see the login form has changed,
after this go to the  Settings->custom-login & put your logo & background image link which you want.
Enjoy.

== Installation ==

1. Upload the folder "wp custom login" to the "/wp-content/plugins/" directory
2. Activate the plugin through the 'Plugins' menu in WordPress
